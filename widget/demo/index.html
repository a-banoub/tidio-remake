Tidio Remake — Widget Demo

This is a fake page on simple1031x.com. The chat widget should appear in the bottom-right corner.

Open DevTools to see the WebSocket connection. The widget connects to ws://127.0.0.1:8080/ws/visitor.

  1. Open this file via the local server (cd widget && npx http-server demo -p 9000) — DON'T file://, that breaks WebSocket.
  2. Make sure the server is running (cd server && npm run dev).
  3. Click the bubble, send a message, watch it land in the server's SQLite DB.
